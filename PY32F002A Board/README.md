# PY32F002A Microcontroller Board

## English

This board uses the PY32F002A, a very low cost 32-bit ARM Cortex M0+ microcontroller from PUYA. With 20 kbytes of Flash, 3 kbytes of RAM and running at 24 MHz, this can be used for many embedded applications.
